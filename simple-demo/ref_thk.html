My One-Week Learning Sprint

1
Day 1-2: Foundation Building

Started with HTML5 semantics and CSS fundamentals. Created basic page structures and learned about responsive design principles.

2
Day 3: Bootstrap Discovery

Discovered Bootstrap's grid system and components. Built my first responsive layout without writing extensive custom CSS.

3
Day 4: JavaScript Immersion

Dived into JavaScript fundamentals, DOM manipulation, and event handling. Added interactivity to static pages.

4
Day 5: SCSS & Advanced Styling

Learned SCSS features like variables, nesting, and mixins. Converted CSS to more maintainable SCSS structure.

5
Day 6: Vite & Build Process

Set up my first Vite project. Learned about modern frontend tooling, modules, and build optimization.

6
Day 7: Portfolio Creation

Integrated all learned technologies to build this comprehensive portfolio. Applied best practices and polished the final product.
